# weatherdashboard

## Description

So I could track the weather from diffrent cities I built the site to save a search history and show the current and 5 day forecast for a city searched.  With a click of the past searched city the current and 5 day forecast will display for that city.  





## Usage
https://cjtavares.github.io/weatherdashboard/

Enter a city and search to see the current and 5 day forecast. If you click on any of the past searched cities the current and the 5 day forecast will show for that city.
